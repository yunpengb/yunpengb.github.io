常用工具

ESPyHarp
是一个可以给MP传输文件的java程序，win下使用，硬伤是无法快速便捷的中断com占用，导致使用起来不方便。
webrepl client html
是一个网页形式的文件传输工具，使用起来比较方便，每次连接一般是要输入密码（根据MP的设置来）.
esptool
这个上面都有说，python 公共库里集成了，使用pip安装
